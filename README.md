# TrackDelay
**Public Transport Delay Reporting Portal**

TrackDelay is a crowdsourced platform that helps commuters detect and report public transport delays using community-driven reports. Users can view live route statuses, while administrators can track delay trends and analytics in real-time.

## Features
- **Live Route Statuses**: Real-time mock statuses of public transport systems.
- **Delay Reporting**: 3-step reporting form for capturing delay reasons and severities.
- **Crowd Aggregation**: Confident aggregations from community reports.
- **Admin Dashboard**: Real-time feed, analytics and charts.
- **Responsive Aesthetics**: Beautiful UI leveraging Glassmorphism and dynamic themes.

## Usage
Currently hosted locally via simple HTTP servers. Simply serve the directory and navigate to `index.html`.

Enjoy tracking!
